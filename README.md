# Drop-$hop Homy$4 Life

Browser-based marketplace app scaffold for Drop-$hop Homy$ 4 Life — React + TypeScript + Vite + PWA starter.

Repository: kaetlynhall2-code/Repo

## Clock Demo

This folder contains a small demo app that displays the current time in multiple time zones.

## Scripts

- npm run dev — start Vite dev server
- npm run build — build production
- npm run preview — preview production build
- npm test — run unit tests (Vitest)

## Setup

1. Install dependencies: npm install
2. Start dev server: npm run dev
3. Open http://localhost:5173

## Notes

- Time zone strings must be valid IANA identifiers (e.g., "America/Los_Angeles").
- The demo uses the Intl API to format times.
